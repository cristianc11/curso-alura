- git clone [URl]: clonar repositorio
- git config --global user.name "": agregar nombre de usuario
- git config --global user.email "": agregar email de usuario
- git confing --list: ver lista de configuraciones
- git log: ver los logs del proyecto
- git log --oneline: ver los logs del proyecto en 1 linea
- git log -p: ver alteraciones de commit
- git log --author="user_name": informacion del autor del commit
- git log --since=1.month.ago --until=1.day.ago: buscar informacion del commit por fecha
- git log --pretty="format:%h %s": formatea la visualizacion del commit
- git pull: actualiza el estado y los archivos del repositorio en mi local
- git status: verificar el estado de mi proyecto
- git add . o git add [name.archivo]: agrega todos los archivos o el archivo para el commit
- git commit -m "comentario": envio de commit con los cambios realizados
- git push: envia los cambios realizados para que se sincronicen en la nube de git
- git diff o git diff [archivo]: muestra los cambios realizados en el proyecto o en un archivo en especifico
- git restore --source #hash [archivo]: Restaura un archivo en especifico (el #hash se consigue en git log --online)
- git checkout -b [nombre de rama]: crea una nueva rama de trabajo y cambia a esa rama
- git branch: nos muestra una lista de las ramas
- git switch [nomre rama]: cambia de rama
- git push origin [rama]: envia los cambios a la nube de git cuando se tienen varias ramas
- git branch [nombre rama]: crea una nueva rama
- git merge [nombre rama]: unifica los cambios realizados desde otra rama
